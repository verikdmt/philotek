Security

Philotek recognizes that the security space is confusing to customers because thousands of vendors with different solutions all say they can do the same thing. The reality is you do need a few products to create a reasonably strong security posture but you don’t need everything that is put in front of you.

Philotek has broken down the security landscape into 4 major components:

  1. End point protection. This is where antivirus software, malware detection and a set of endpoint and mobile controls protects the initial point of entry.

  2. Avenue of attack in the threat landscape is the traditional attack surface of your network. This is where you would typically provide protection via firewalls, identity management tools, email protection, and potentially the use of cloud and workspace protection tools.

  3. The next level of security that Philotek assists customers with evaluating and architecting is the optimization of a customer’s Security Operations. This can be done in a number of ways, but it primarily consists of hunting for threats using tools that detect anomalies and then responding to these anomalies by either considering them false positives or acting on them to remediate the issue. Philotek can provide this as either a managed service or by recommending tools that can be used in this process.

  4. Ultimately, even the best laid plans can have weaknesses due to user error, hidden malicious code in traditionally benign software or other attack vectors that continually evolve. Because of this Philotek has partnered with several digital forensic firms and created our own set of common protocols to use for incident response and management. This coupled with our compliance and data governance offerings helps provide customers the ability to manage an incident should it occur even though security protocols were properly established.
